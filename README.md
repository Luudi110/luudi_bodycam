# luudi_bodycam
Luudi Bodycam er et avanceret bodycam-script, der giver et realistisk bodycam-overlay på skærmen og mulighed for at se andre spilleres bodycams samt køretøjers dashcams i realtid. Scriptet understøtter optagelse af korte videoklip, som gemmes sammen med detaljerede spillerlogs til senere gennemsyn.
